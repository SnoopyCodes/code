timer for speed gaming in red rising
tracks resource values and end of game with house rules

commands:
type enter after a command to activate it
p -> pause the game (press p to start as well)
space -> next player turn
u -> undo last action (not implemented)

to alter a player's balance 
[1-6]['f', 'h', 'i'][+-][0-9]
1-6 is id of player in order of table shown
+- is optional
f -> fleet, h -> helium, i-> institute
+-[0-9] is change, +- is optional and will default to positive

you can stack as many changes as you want, and gain sovereign s goes at the end
1f-23h+1s = player 1 loses 2 fleet positions and player 3 gains 1 helium
the current player gains sovereign


(relevant) house rules:
game ends at 7/7 or 10/10 or 10/10/10
fleet/influence can extend past 10
no taking from deck after deploying a card
game ends instantly after 7/7 10/10 10/10/10 (+1 turn for apollo)
if rising die gives resource bonus it is auto added

timer:
point reduction per player p is 
(player with most second's time in seconds - p's time in seconds) / 2
